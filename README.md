# MyReactWorkspace_codespaces

This repo is to learn and practice React topics

## bg-changer project (React with tailwind css)

> A simple project where the background color changes when color buttons are clicked. This project was created to better understand the `useState` hook in React.

## passwordGenerator Project

> A simple project to generate passwords with customizable options. Users can choose a password length between 8 and 30 characters and select options to include numbers and special characters. This project was created to understand the use of the `useCallback` and `useRef` hooks in React.

## currencyConvertor Project

> A simple project to convert currencies. This project was created to understand how to build a custom hook and how to use the `useId` hook in React.

## miniContext & themeSwitcher Project

> Both of the projects are created to understand how context api handle state management. `useContext`, `createContext` and **Context Provider** are the topics covered in this two projects.
